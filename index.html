Privacy Policy

Product: Gmail Follow-up Tracker
Effective date: April 14, 2026

Gmail Follow-up Tracker helps users turn Gmail threads into follow-up tasks with recurring reminders. This policy explains what data is processed and how it is used.

1. Data We Process

To provide core functionality, the extension may process:

  • Email thread context visible on Gmail pages (for example: subject, sender, thread identifier, thread URL).
  • Task metadata (status, reminder interval, reminder timestamps, completion state).

2. How We Use Data

  • Create and manage follow-up tasks for Gmail threads.
  • Schedule recurring reminders.
  • Display browser notifications and badge counts for due tasks.

3. Storage, Sharing, and Selling

  • Data is stored locally in the user’s browser via chrome.storage.local.
  • We do not sell user data.
  • We do not share user data with third parties.
  • We do not use data for advertising.

4. Network and Remote Code

  • This extension does not rely on a backend service for core features in the current version.
  • All executable code is packaged with the extension.
  • No remotely hosted code (remote JavaScript/WASM execution) is used.

5. Permissions Explanation

  • storage: Save tasks and reminder settings locally.
  • alarms: Trigger recurring reminder checks.
  • notifications: Show follow-up reminder notifications.
  • tabs: Open related Gmail thread from notifications and query active tab context.
  • activeTab: Access the active Gmail tab when user interacts with the extension.
  • scripting: Inject/re-inject content script in Gmail tab when needed.
  • https://mail.google.com/* (host permission): Run only on Gmail pages.

6. User Control

Users can remove tasks at any time, mark tasks as done, disable notifications in browser settings, or uninstall the extension to stop all processing.

7. Children’s Privacy

This extension is not directed to children under 13 and is intended for productivity use by general users.

8. Changes to This Policy

We may update this policy from time to time. Updated versions will be posted at this URL with a new effective date.
