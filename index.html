Privacy Policy for Gmail Follow-up Tracker Gmail Follow-up Tracker processes Gmail thread information (such as subject, sender, and thread context) to provide follow-up task tracking and recurring reminders. Data Collection and Storage - The extension stores follow-up task data locally in the user's browser using chrome.storage.local. - The extension does not sell user data. - The extension does not share user data with third parties. - The extension does not use user data for advertising. Permissions Usage - storage: save follow-up tasks and reminder settings locally. - alarms: schedule recurring reminder checks. - notifications: display due follow-up reminders. - tabs and activeTab: access the active Gmail tab and open related threads. - scripting and host permission (https://mail.google.com/*): inject and run extension functionality on Gmail pages.
